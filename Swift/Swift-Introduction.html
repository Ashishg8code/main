Introduction to Swift

  • Swift is a newly tool developed by Apple Inc. which is fast,small and static compile language to build products for iOS,OS X, watchOS, tvOS, and Linux.

  • Development on Swift was begun in July 2010 by Chris Lattner (main author of LLVM) , with the eventual collaboration of many other programmers at Apple.

  • Swift is designed to work with Apple's Cocoa and Cocoa Touch frameworks and it is built with the LLVM compiler framework.

  • Swift is build with Optimizer and Autovectorizer,with ARC memory management.

  • Since the language is open-source, there are prospects of it being ported to the web. Some web frameworks have already been developed, such as Kitura by IBM and the open-source Perfect.

  • Google is said to be considering using Swift as a first class language for its operating system Android.
