Types

  • Variable and constants have a data type. The type describes the nature of the data and provides information to the compiler on how to handle the data. Based on the type of a constant or variable,the compiler knows how much memory to reserve and will also be able to help with the type checking.

    • Let's see this in Action. Suppose you want to model a small town in your code. You might want a variable for the number of trafficLights

     import Cocoa

 var numberOfTrafficLights = "two"

    Here,Cocoa is a framework, in the next line, we have a variable numberOfTrafficLights declared by Swift's keyword var whose value is assigned a type of String "two".

    Adding "two" and 2 
     import Cocoa

 var numberOfTrafficLights = "two"

 numberOfTrafficLights+=2

    The compiler will give an error,saying adding String to Int is not allowed.

    To get rid of this issue, you can declare the variable as an integer.

     import Cocoa

 var numberOfTrafficLights : Int = 4

 numberOfTrafficLights+=2

    Now,you explicitly declaring the variable to be of Int type using Swift's type annotation syntax.The colon represents " ... of type... ",so code can be read as :"Declare a variable called numberOfTrafficLights of type Int that starts out with a value of 2 .

    Declaring Constants

    Constants in Swift are declared using Swift's reserved keyword let 

     import Cocoa

 let numberOfTrafficLights : Int = 4

    String Interpolation

    String Interpolation allows you to combine constants and variables values in a String.

     import Cocoa

 let numberOfTrafficLights : Int = 4
 var populationOfTown : Int
     populationOfTown = 400
  let nameOfTown = "MTOWN"
  let description = "\(nameOfTown) has a population of \(populationOfTown) and having \(numberOfTrafficLights) traffic lights."
  print(description)

    The code gives output:

     MTOWN has a population of 400 and having 4 traffic lights.

    The \() syntax represents placeholder in the String literal which accesses an instance's value and places it within the new String and offcourse, print() is a function of Swift for printing strings to console.

    Conditionals


    if/else

    if else condtion executes on a specific condition in code. 

     import Cocoa

 var population : Int = 1000
 var message : String

 if population < 1000
 {
    message = "\(population) is a small town !"
 }
 else
 {
    message = "\(population) is big town !"
 }

 print(message)

    Here you have checked if population is less than 1000 , then based on the outcome, the if or else block is to be executed, the String interpolation is used to take the message.

    is there a theatre ? 
     import Cocoa

 var population : Int = 1000
 var message : String
 var isTheatre : Bool = true

 if population < 1000
 {
    message = "\(population) is a small town !"
 }
 else
 {
    message = "\(population) is big town !"
 }

 print(message)

 if !isTheatre
 {
   print("Where do we watch movie?")
 }

    Here you have added a new variable hasTheatre as boolean which is set to true.Boolean types can take values only true or false.

    ! is logical not, so you can think ! as inventing true to false and false to true, so in this case the print function will not execute as we have already set it to true.

    Ternary operator

    Ternary operator is very similar to if/else, but has more concise syntax.

    The syntax looks like a?b:c,reads like if a is true,then do b else do c.

     import Cocoa

 var population : Int = 1000
 var message : String
 var isTheatre : Bool = true

 message = population < 1000 ? "\(population) is a small town !" : "\(population) is big town !"

 print(message)

 if !isTheatre
 {
   print("Where do we watch movie?")
 }

    Ternary operator is great for concise statements, but if your stratements starts wrapping to next line,you can be using if/else conditionals.

    Nested ifs

    You can use other if statements in an if, to check more thant two possibilities.

     import Cocoa

 var population : Int = 1000
 var message : String
 var isTheatre : Bool = true

 if population < 1000
 {
    message = "\(population) is a small town !"
 }
 else
 {
    if population >= 500 && population <= 900
    {
    message = "\(population) is medium town !"
